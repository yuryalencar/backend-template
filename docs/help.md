# API Documentation

## Endpoints

- `GET /users`: Retrieve all users
- `GET /users/:id`: Retrieve user by ID
- `POST /users`: Create a new user
- `PUT /users/:id`: Update user by ID
- `DELETE /users/:id`: Delete user by ID
